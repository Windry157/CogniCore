# 🚀 Nexus 界面 - 快速测试指南

## 📋 前置条件

1. 确保服务正在运行
2. 如果未运行，请先启动服务

## 🔄 启动步骤

### 方法 1：使用 start_portable.bat（推荐）

1. 进入 `cognicore` 目录
2. 双击 `start_portable.bat`
3. 等待服务启动完成

### 方法 2：手动启动

```bash
cd cognicore
python app.py
```

## 🌐 访问 Nexus 界面

服务启动后，在浏览器中打开：

```
http://localhost:8002/nexus
```

或

```
http://127.0.0.1:8002/nexus
```

## 🔧 故障排除

### 问题：无法访问 /nexus

**解决方案**：
1. 确认服务是否正常启动
2. 检查端口是否正确（默认 8002）
3. 确认 `cognicore/frontend/nexus.html` 文件存在

### 问题：页面可以打开但功能不正常

**解决方案**：
1. 打开浏览器开发者工具（F12）
2. 查看 Console 是否有错误
3. 检查 Network 标签中的请求

### 问题：点击发送后无反应

**解决方案**：
1. 这是演示版本，响应是模拟的
2. 真实功能需要连接完整的后端 API
3. 查看 `/docs` 了解真实 API 用法
