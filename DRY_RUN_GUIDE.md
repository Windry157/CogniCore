# 🎯 CogniCore-Portable 部署模拟演练指南

**演练目标**：在接近生产环境的条件下，完整验证部署、运行、回滚全流程  
**预计时间**：30-60 分钟  
**版本**：1.0 - 2026-05-23

---

## 📋 演练前准备

### 1.1 环境准备

| 准备项 | 说明 | 状态 |
|--------|------|------|
| 演练环境机器 | Windows 或 Linux | ☐ |
| Python 3.8+ 已安装 | `python --version` 验证 | ☐ |
| 磁盘空间 | 至少 100MB 可用 | ☐ |
| 备份目录 | 用于存放测试备份 | ☐ |

### 1.2 交付物准备

确认以下文件存在于演练目录：
- `cognicore/` - 完整应用目录
- `RUNBOOK.md` - 运维手册
- `DEPLOYMENT.md` - 部署计划
- `FINAL_ACCEPTANCE.md` - 验收报告

## 🚀 演练步骤（共 10 步）

---

### 【步骤 1】初始部署（10 分钟）

**目标**：验证从零开始的部署流程

| 步骤 | 操作 | 预期结果 | 验证人 | 状态 |
|------|------|----------|--------|------|
| 1.1 | 进入 `cognicore/` 目录 | 目录存在 | | ☐ |
| 1.2 | 双击 `start_robust_pipeline.bat` | 无错误，测试运行 | | ☐ |
| 1.3 | 检查控制台输出 | 看到 `✅ 工业级数据管道测试成功` | | ☐ |
| 1.4 | 验证数据文件 | `data/pipeline.db` 已创建 | | ☐ |

---
